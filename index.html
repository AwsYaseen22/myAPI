Home consumed stuff API

To use the api please head to the following url:

  • Main page (this page):
    • https://consumed-stuff-api.herokuapp.com

  • List all items:
    • https://consumed-stuff-api.herokuapp.com/api

  • Get specific item by name:
    • https://consumed-stuff-api.herokuapp.com/api/items/Milk

  • Add item to the itms list:
    • Post to the url with the data at the body: https://consumed-stuff-api.herokuapp.com/api/items

  • List all categories
    • https://consumed-stuff-api.herokuapp.com/api/categories

  • List all items filtered by category:
    • https://consumed-stuff-api.herokuapp.com/api/items-by-category/:category
  • Add new category:
    • Post to the url with data in the body with json format: https://consumed-stuff-api.herokuapp.com/api/categories
